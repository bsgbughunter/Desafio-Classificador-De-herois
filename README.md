> Um projeto de lógica de programação para classificar heróis com base em sua experiência (XP).

##  Sobre o Projeto

Este projeto foi desenvolvido como parte de um desafio de lógica de programção DIO - SANTANDER . O objetivo principal foi consolidar conhecimentos sobre **variáveis**, **operadores**, **estruturas de decisão** e **entrada de dados** via terminal.

O programa recebe o nome e a quantidade de experiência (XP) de um herói e, através de uma estrutura de decisão, retorna o nível em que ele se encontra (Ferro, Bronze, Prata, etc.).

##  Tecnologias Utilizadas

* **JavaScript:** Linguagem principal utilizada para a lógica.
* **Node.js:** Ambiente de execução para rodar o JavaScript fora do navegador.
* **Biblioteca `readline`:** Módulo nativo do Node.js usado para capturar as entradas do usuário (Input) no terminal.

---

##  Funcionalidades

- [x] **Interação via Terminal:** O usuário insere dados diretamente no console.
- [x] **Validação de Tipos:** Conversão de entrada de texto (String) para número (Int) para comparação de XP.
- [x] **Estrutura de Decisão Complexa:** Utilização de `switch/case` com condições booleanas (`switch(true)`) para verificar faixas de valores de XP.
- [x] **Saída Dinâmica:** Exibição de mensagem personalizada com o nome do herói e seu nível correspondente.

---

##  Como Rodar o Projeto

1. Certifique-se de ter o **Node.js** instalado em sua máquina.
2. Clone este repositório ou baixe os arquivos.
3. Abra o terminal na pasta do projeto.
4. Execute o comando:


node classificadrDeHeroi.js
Siga as instruções no terminal digitando o nome e o XP do herói.
